# Lush Engineering Website (Paste-Ready)

## Included
- index.html (hero image + animated hero text)
- about.html
- services.html (hero image)
- projects.html (filters + lazy-loaded gallery + lightbox)
- contact.html (WhatsApp form)
- assets/js/main.js (menu, year, lazy-load, filters, lightbox, WhatsApp form)
- assets/img/placeholder.jpg (used for lazy-loading)

## IMPORTANT: Add your images
Place these files inside `assets/img/` in your repo (same folder as placeholder.jpg):
- logo.svg
- favicon.png
- og-image.jpg
- hero.jpg
- page1_img1.jpeg, page1_img2.jpeg, page1_img3.jpeg, page1_img5.jpeg, page2_img1.jpeg, page4_img2.jpeg
  (or update filenames in `projects.html`)

## Run locally
`python -m http.server 8000`
Open: http://localhost:8000

## Deploy
Works on GitHub Pages / Netlify / any static host.
